# LovePrint Scoring System

## Core Dimensions
1. **Stability** — long-term potential, structure, success, responsibility.
2. **Connection** — communication, emotional depth, understanding.
3. **Attraction** — physical appeal, presentation, lifestyle appeal.
4. **Chemistry** — passion, spark, intensity, sexual energy.

## Trait Mapping
| Trait | Stability | Connection | Attraction | Chemistry |
|---|---:|---:|---:|---:|
| Masculine / Confidence | 2 | 0 | 0 | 1 |
| Conversation | 0 | 3 | 0 | 0 |
| Fit | 0 | 0 | 2 | 1 |
| Smells Good | 0 | 0 | 2 | 0 |
| Good D. / Sexual Chemistry | 0 | 0 | 0 | 3 |
| Successful / Ambitious | 3 | 0 | 0 | 0 |

## Bias Detection
- **Long-Term Bias:** Stability + Connection dominate.
- **Mixed Bias:** Scores are balanced.
- **Short-Term Bias:** Attraction + Chemistry dominate.

## Metrics
### Long-Term Success Score
`(Stability + Connection) / Total Score * 100`

### Attraction Bias Score
`(Attraction + Chemistry) / Total Score * 100`

### Volatility Score
`Chemistry * 2 - Stability`
