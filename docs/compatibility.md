# Compatibility Report

## Score Calculation
Compare each person’s scores across:
- Stability
- Connection
- Attraction
- Chemistry

Formula:
`100 - (Total Dimension Difference × 5)`

## Compatibility Types
1. **Power Couple** — high stability and connection.
2. **High Chemistry Match** — strong attraction and chemistry.
3. **Growth Challenge** — misaligned but fixable.
4. **Short-Term Spark** — exciting but unstable.

## Report Sections
1. Compatibility score
2. Quick summary
3. Where you align
4. Where you clash
5. Relationship dynamic
6. Hidden tension
7. What each person needs
8. Future prediction
9. Upgrade CTA

## Example
**You:** Growth Partner  
**Partner:** Chemistry Chaser  

**Compatibility Score:** 62%

**Summary:** Strong attraction with different long-term priorities.

**Hidden Tension:** One person is building a future. The other is chasing a feeling.

**Prediction:** High passion, medium-term potential, long-term uncertainty.
